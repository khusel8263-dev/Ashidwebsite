# Center Navigation Template Package

## Ашиглах

1. ZIP-ийг extract хийнэ.
2. `index.html` файлыг browser дээр нээнэ.
3. Өөрийн project дээр ашиглах бол `index.html`, `css/`, `js/`, `images/`, `fonts/`, `revolution/` folder-уудыг хамтад нь хуулна.

## Юу зассан бэ?

- `center-navigation.html`-ийн asset folder structure-ийг хадгалсан.
- Remote `placehold.it` зургуудыг local placeholder зургаар сольсон.
- Dependency path эвдрэхээс сэргийлж original нэртэй CSS/JS файлуудыг үүсгэсэн.
- Header mobile toggle, smooth scroll, scroll-top зэрэг basic interaction ажиллана.

## Exact original ThemeForest package гаргах

Чиний computer дээр original folder байгаа бол `tools/pack-original-from-local.ps1` script-ийг ажиллуулаад яг original `css/js/images/fonts/revolution` файлуудтай package үүсгэж болно.
