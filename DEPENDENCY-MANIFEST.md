# Center Navigation Template Dependency Manifest

Source page inspected: `html/center-navigation.html` from `khusel8263-dev/themeforest`.

This package preserves the same folder/file paths used by the page:

- `css/` — stylesheet paths from the original template
- `js/` — script paths from the original template
- `images/` — favicon, logo, banner and local placeholder images
- `fonts/` — reserved folder for icon fonts
- `revolution/` — Revolution slider CSS/JS paths

Important: the original HTML used remote `http://placehold.it/...` images. They were replaced here by local SVG placeholder assets under `images/placeholders/` so the UI does not show broken remote images.

Open `index.html` directly in a browser or serve this folder through any static web server.
